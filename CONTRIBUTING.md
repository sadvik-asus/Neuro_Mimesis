# Contributing to Neuro-Mimesis

First off, thank you for considering contributing to Neuro-Mimesis! It's people like you that make it a great tool.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if the bug has already been reported.
* If not, create a new issue. Please include details of your OS, Python version, and a clear description of the problem.

### Suggesting Enhancements
* We are always looking for ways to improve the Trust Score algorithm and Active Defense mechanisms.
* Feel free to open an issue or start a discussion about new features.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code lints.
6. Issue that pull request!

## Code Style
* **Frontend**: Follow the standard React/TypeScript patterns. Use Tailwind CSS for all styling.
* **Backend**: Follow PEP 8 for Python code.
* **Security**: All security-sensitive code should be well-commented and follow best practices for handling OS-level hooks.

## GSoC Aspirants
If you are contributing as part of a GSoC application:
* Focus on the **Roadmap** items.
* Show depth in your understanding of the "Behavioral Biometrics" aspect.
* High-quality documentation is just as valuable as code!
